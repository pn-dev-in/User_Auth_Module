# User Authentication Module

## Description
This project implements a secure user authentication module as part of an internship assessment.
It includes user registration, login, password change, and profile management features.

## Features
- User Registration
- User Login
- Change Password
- View & Update User Profile
- Secure password hashing
- JWT-based authentication

## Tech Stack
- Programming Language: JavaScript
- Backend Framework: Node.js with Express.js
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Authentication: JWT
- Password Security: bcrypt

## Security Practices
- Passwords are hashed using bcrypt
- JWT is used for authentication and authorization
- Protected routes for profile and password change
- No plaintext password storage

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start the server using `node server.js`
4. Open browser at `http://localhost:4000`
